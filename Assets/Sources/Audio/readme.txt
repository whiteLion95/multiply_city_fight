1. Stickman collisions - столкновение стикманов разных цветов, звук воспроизводится случайно один раз
2. lose - воспроизводится в случае, если игрок потерял последний город один раз
3. tap - воспроизводится при тапе один раз 
4. win - воспроизводится при появлении ui победы один раз
5. захват города - воспроизводится один раз при захвате игроком города другого цвета
6. захват клетки - воспроизводится один раз при построении пути через клетку
7. потеря города - воспроизводится единожды в случае, если бот захватил город игрока, на нейтральные города или города других цветов не работает
8. столкновение стикмана с городом - при коллизии стикмана с любым городом единожды для каждого стикмана
9. удаление пути - воспроизводится при удалении пути единожды 
10. умножение стикмана - воспроизводится при коллизии стикмана и клетки-множителя единожды для каждого входящего стикмана
